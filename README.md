# 基于知识图谱的新疆民族文化大数据平台

#### 介绍
{
    基于知识图谱的新疆民族文化大数据平台是一个纯前端项目。
    利用的技术为html、css、js。
    主要分为三个界面
    1.欢迎界面
    2.各少数民族介绍界面
    3.新疆少数民族知识图谱界面
    其中知识图谱的可视化利用echarts库，数据存储在echarts.js中，同时利用mysql.js可以将相关的知识图谱节点转化为sql语句，插入mysql中
}
